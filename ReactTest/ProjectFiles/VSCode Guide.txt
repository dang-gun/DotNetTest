1. 필수 설치
확장 : C# - https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp
SDK : .NET 6.0 SDK - https://dotnet.microsoft.com/en-us/download/dotnet/6.0

--닷넷 버전 확인
dotnet --info 



2. 프로젝트 로드하기
터미널을 열고 작업한다.

1) 경로 이동
- "*.csproj"파일이 있는 경로로 이동
- 프로젝트를 연상태로 터미널을 열면 대상 경로이다.

cd [프로젝트 경로]
e.g. D:\work\project\vs\DotNetSamples\ToVsCodeTest



2) 프로젝트 로드
- '실행 및 디버그' 창에서 "디버그하거나 실행할 수 있는 파일 열기입니다."라는 메시지가 나오면 할필요 없다.

code -r [프로젝트 명]
e.g. code -r ToVsCodeTest

빌드 및 디버그 자산이 없다는 메시지가 나오면 "예"를 눌러 추가한다.


3) 'launch.json' 생성
- 해당 파일이 없으면 '실행 및 디버그'창에서 'launch.json 파일 만들기'를 눌러 생성할 수 있다.
-> 'launch.json 파일 만들기' -> 디버거 선택 -> .NET 5+ and Core 를 선택한다.



3. 프로젝트 실행

1) 실행 및 디버그 버튼 누르기

2) 디버깅 없이 실행
Ctrl+F5

3) 인증서가 없음 오류가 나면
dotnet dev-certs https --trust
명령을 사용하여 생성한다.
